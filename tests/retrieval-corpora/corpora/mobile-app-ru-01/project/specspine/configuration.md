# Конфигурация

В сборку входят адрес API, идентификатор окружения и публичный ключ проверки.
Удалённая конфигурация содержит интервалы обновления, лимиты кэша, рекламные
параметры и значения по умолчанию для [флагов функций](feature-flags.md).

Секреты и пользовательские токены в конфигурацию не входят. Изменения
применяются после проверки схемы и сохраняются до следующего успешного
обновления. Параметры могут задавать частоту фоновой синхронизации, но не
правило разрешения конфликтов. Они могут ограничить показ уведомлений, но не
определяют ротацию push-токена.
