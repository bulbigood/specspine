# Доступность интерфейса

Этот документ владеет адаптацией к системному масштабу текста.

Текст масштабируется до 200 процентов без обрезания смысловых подписей.
Контейнеры растут по содержимому, а горизонтальная прокрутка для основного
текста запрещена. При крупном шрифте панель действий переносится в вертикальную
компоновку. Иконка без видимой подписи обязана иметь имя для экранного диктора.

Компоненты проверяются при коэффициентах 1,0; 1,5 и 2,0. Карта может сохранять
размер графики, но её элементы управления и подсказки масштабируются.
Расположение экранов описывает [навигация](navigation.md).
