# Флаги функций

Флаги загружаются после базовой [конфигурации](configuration.md) и имеют
встроенные безопасные значения. Новый экран включается только для разрешённой
версии приложения. Изменение флага не удаляет локальные данные и не меняет
формат базы.

Оперативный выключатель может скрыть вход в функцию или остановить создание
новой работы. Порядок действий во время сбоя задаёт
[реагирование на инциденты](incident-playbook.md). Флаг не заменяет правила
оплаты, синхронизации или регистрации уведомлений.
