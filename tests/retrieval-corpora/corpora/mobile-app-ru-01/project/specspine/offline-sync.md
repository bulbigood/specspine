# Автономная синхронизация

Этот документ владеет правилами разрешения конфликтов при синхронизации.

Для избранного маршрута побеждает изменение с большей серверной ревизией.
Если ревизии равны, удаление имеет приоритет над редактированием. Клиент не
сливает поля и не сравнивает время устройства. После разрешения конфликта
намерение помечается завершённым в той же транзакции, где записан ответ.

Формат очереди и транзакции принадлежат [локальному хранилищу](local-storage.md).
Повторную авторизацию описывает [аутентификация](authentication.md).
