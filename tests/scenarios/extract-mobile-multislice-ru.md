# Сценарий: возврат из банка и внешняя ссылка

## User request

```text
Спроектируй возврат из банковского приложения: отдельно найди правило, когда
покупка действительно подтверждена и билет можно показать, и отдельно правила
разбора внешней ссылки на билет до передачи в навигацию. Подготовь минимальный
архитектурный context handoff, используй только SpecSpine как источник истины и
не изменяй файлы проекта.
```
